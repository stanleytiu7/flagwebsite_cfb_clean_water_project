Welcome CRWA Staff and Interns

Add a CSO or Cyanobacteria event

  1. First enter the CSO or Cyanobacteria event for the appropriate boathouse
  2. Then run the script to update the flagging page

Update or end a CSO or Cyanobacteria event

  1. First look up the ID number for the event you want to update.
  2. Then update or delete the events.
Note: You must update the cyanobacteria event for each boathouse individually.
